# JMeterAdvancedDemo
This project describes how to create a reusable JMeter script that takes into consideration the following aspects:
1- Reusability
2- Manintainability 
3- Extention

We have used the following components to achive our target:
1- Simple controller
2- Transaction controller
3- Module controller
4- Test fragement
